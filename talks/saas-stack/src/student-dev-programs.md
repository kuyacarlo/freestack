# Programs & Free Tiers — Who Gets What

Companion to `main.md` (SaaS free tiers) and the student pack deep-dives below.

Legend for every table:

| Column | Meaning |
| :--- | :--- |
| **Cost** | `Free` = $0 while eligible · `Discount` = pay less · `Credits` = prepaid $ burn · `Paid` = normal pricing |
| **Student?** | `Required` · `Helps` (better deal if verified) · `No` (anyone) |
| **Commercial?** | Can you use it for a startup / paid product? `Yes` · `No / Edu-only` · `Hobby-only` · `Check ToS` |

> Offers rotate. Re-verify student status every 1–2 years. Edu licenses are usually **non-commercial**.

## Quick picker

| You are… | Start here |
| :--- | :--- |
| **Student** | GitHub Pack → JetBrains → Azure $100 → Figma Edu → school MATLAB/Autodesk → free SaaS stack |
| **Not a student** | Free SaaS forever-tiers (`main.md`) → Oracle Always Free → open-source self-host → startup credits (AWS Activate, etc.) |
| **Student building a startup** | Use **commercial-OK free tiers** for prod; keep Edu tools for learning only |
| **Graduate / lost verification** | Export data before lapse; move to free tiers + JetBrains continuing discount |

### Non-student / alumni, $0–low stack

| Layer | Pick |
| :--- | :--- |
| Editor | VS Code + Copilot paid *or* Cursor Hobby |
| Host | Cloudflare / Render / Koyeb |
| DB | Neon + Upstash |
| Auth | Clerk free |
| Design | Figma Starter |
| Storage | R2 |
| Email | Resend free |
| Observe | PostHog + Sentry free |

### Student shipping a **real** product

Use **commercial-OK** free tiers for production (Neon, Cloudflare, Clerk, Resend…).  
Keep JetBrains Edu / Autodesk / MATLAB / Figma Edu for schoolwork only — buy commercial seats when money comes in. Vercel Hobby is **not** for commercial; budget Pro.
