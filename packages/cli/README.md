# @kuyacarlo/freestack

CLI for [freestack](https://freestack.kuyacarlo.dev) claim checklists and tool lookup.

```bash
pnpm dlx @kuyacarlo/freestack claim --student --commercial
pnpm dlx @kuyacarlo/freestack tools --category ai --commercial yes --limit 10
pnpm dlx @kuyacarlo/freestack tool neon --json
pnpm dlx @kuyacarlo/freestack health
```

Set `FREESTACK_API` to hit a local preview (`http://127.0.0.1:4321`) instead of production.
